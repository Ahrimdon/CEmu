Download Development Builds

Note that some of these builds are bleeding-edge versions - they may break at any time! Recommended only if you want to try new features, and don't mind occasional bugs.

For now, this is the version that will allow use to use "USB transfers", i.e. being able to transfers files such as OSes and apps to the emulated calculator.

The Windows builds are provided by @jacobly0 (on his server, built with MinGW). The Linux and macOS builds are provided by @adriweb (built by OBS for Linux).

About CEmu

CEmu is a third-party cross-platform TI-84 Plus CE / TI-83 Premium CE calculator emulator, focused on developer features. Any amount would help support and continue development on this project and is greatly appreciated.
